# RayCaster Engine

A minimal open-source raycasting engine written in C, developed as a final project
for the Computer Workshop course.

## Installation

This project is intended to be built and run on Linux.

Required dependencies:
- gcc
- make
- raylib

Install dependencies on Ubuntu:

sudo apt update
sudo apt install build-essential git libraylib-dev

Clone the repository:

git clone https://github.com/AshkanEs-eng/my-raycaster-launch.git
cd my-raycaster-launch

Build and run:

make
./raycaster
